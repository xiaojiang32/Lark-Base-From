# 表单对齐方式配置优化

## 一、需求分析

1. **删除全局表单对齐配置**：从配置面板中移除"表单对齐"选项
2. **为组件添加对齐方式配置**：
   - 支持左对齐、居中对齐、右对齐
   - 组合布局不需要支持对齐方式
   - 每个组件可以独立配置对齐方式

## 二、实现计划

### 1. 修改数据模型

**`src/types.ts`**：
- 添加 `FormAlign` 类型（如果不存在）
- 在 `IFormField` 接口中添加 `align` 属性
- 从 `IFormConfig` 接口中移除 `align` 属性

### 2. 修改配置面板

**`src/components/Form/index.tsx`**：
- 从 `GlobalConfigPanel` 中移除表单对齐配置
- 删除 `getFormAlign` 函数
- 修改表单容器，移除 `alignItems: getFormAlign()` 配置

**`src/components/Form/config/FieldConfigPanel/index.tsx`**：
- 在字段配置中添加对齐方式选项
- 组合组件不显示对齐方式配置

### 3. 修改组件渲染

**`src/components/Form/index.tsx`**：
- 修改 `renderField` 函数，为每个字段添加对齐样式
- 组合组件保持原有逻辑

**`src/utils.ts`**：
- 修改 `getDefaultFormConfig` 函数，移除 `align` 配置

## 三、详细实现步骤

1. **修改 `src/types.ts`**
   - 添加 `align?: FormAlign` 到 `IFormField` 接口
   - 移除 `IFormConfig` 中的 `align` 属性

2. **修改 `src/components/Form/index.tsx`**
   - 移除 `GlobalConfigPanel` 中的表单对齐配置
   - 删除 `getFormAlign` 函数
   - 修改表单容器的 `alignItems` 为固定值
   - 在 `renderField` 函数中添加对齐样式

3. **修改 `src/components/Form/config/FieldConfigPanel/index.tsx`**
   - 在 `renderCommonConfig` 函数中添加对齐方式配置
   - 添加条件判断，组合组件不显示对齐方式

4. **修改 `src/utils.ts`**
   - 从 `getDefaultFormConfig` 中移除 `align` 配置

## 四、预期效果

- 全局配置中不再显示表单对齐选项
- 每个非组合组件可以独立配置对齐方式
- 组合组件保持原有逻辑，不受对齐方式影响
- 表单布局更加灵活，支持组件级别的对齐控制

## 五、技术要点

- 使用 CSS `text-align` 或 `align-items` 实现组件对齐
- 在 `renderField` 函数中动态应用对齐样式
- 确保组合组件不受对齐配置影响
- 保持向后兼容性，未配置对齐方式的组件使用默认对齐

## 六、测试要点

- 验证全局配置中不再显示表单对齐选项
- 验证每个非组合组件可以配置对齐方式
- 验证组合组件不显示对齐方式配置
- 验证组件的对齐样式正确应用
- 验证表单提交功能不受影响
- 验证组合组件的子组件对齐正常

通过以上步骤，我们可以实现从全局表单对齐配置到组件级对齐配置的转变，让表单布局更加灵活，同时保持组合组件的原有逻辑不变。
