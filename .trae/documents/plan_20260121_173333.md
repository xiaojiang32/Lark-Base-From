# 表单配置面板布局优化方案

## 📋 优化目标

将 GlobalConfigPanel 的配置项按照功能分组，并使用一行2列的网格布局，使配置面板更加紧凑和清晰。

## 🎯 具体实现步骤

### 1. 修改 GlobalConfigPanel 组件结构

在 `src/components/Form/index.tsx` 中修改 `GlobalConfigPanel` 函数：

**基础信息区**（一行2列）：

* 表单标题、表单描述（并排）

* 表单宽度、表单对齐（并排）

* 字段间距（单独一行）

**样式区**（一行2列）：

* 显示边框、显示阴影（并排）

* 背景颜色、圆角大小（并排）

**提交按钮区**（一行2列）：

* 按钮文本、成功提示（并排）

* 提交后操作、二次确认（并排）

### 2. 更新样式文件

在 `src/components/Form/style.scss` 中添加新的样式类：

```scss
.config-group {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 12px;
  margin-bottom: 12px;
}

.config-group-title {
  grid-column: 1 / -1;
  font-size: 18px;
  font-weight: 600;
  margin: 0 0 8px 0;
  color: var(--ccm-chart-N900);
  padding-bottom: 8px;
  border-bottom: 1px solid var(--divider);
}
```

### 3. 调整间距和样式

* 配置项之间的间距：12px（网格 gap）

* 配置区内边距：10px（保持不变）

* 配置区圆角：4px（保持不变）

* 输入框边框颜色：#d9d9d9（保持不变）

## ✨ 预期效果

* 配置面板更加紧凑，信息密度更高

* 相关配置项并排显示，提高配置效率

* 保持良好的视觉层次和可读性

* 支持暗色模式
